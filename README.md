# Reward Countdown App - Website

This is the official website for the Reward Countdown App, a simple tool designed to help credit card users track reward expiration dates.

## About

Reward Countdown helps credit card enthusiasts manage multiple reward programs, particularly hotel free night certificates from cards like Chase, Amex, Hilton, and Marriott. Never let valuable rewards expire again!

## Website

The website is deployed via GitHub Pages and provides:
- Information about the app and its features
- Target audience and use cases
- Contact information

## Development

The website is a simple, single-page HTML/CSS application with no build process required.

### Local Development

To run locally:
```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Deployment

The site automatically deploys to GitHub Pages via GitHub Actions when changes are pushed to the `main` branch.
